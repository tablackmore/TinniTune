# AURUM — Mixer UI Kit

A high-fidelity recreation of the AURUM two-deck mixer surface, composing the design-system primitives (`Knob`, `Fader`, `JogWheel`, `VUMeter`, `Waveform`, `TransportButton`, `Button`, `Toggle`, `Badge`).

- **index.html** — the full interactive mixer view. Deck A (gold) / Deck B (ice), center channel strip with EQ + faders + crossfader, master section, and a library footer.

This is a visual/interaction recreation — audio is simulated (level meters animate, playheads advance, jog wheels spin). It demonstrates component coverage and layout, not a real audio engine.
